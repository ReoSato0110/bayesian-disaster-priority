# 災害支援優先度・参考指標デモアプリ

⚠️ **重要な注意事項**

このアプリは **災害支援における優先度判断の参考指標** を示す  
**デモ・学習目的のアプリケーション**です。

- 意思決定を自動化するものではありません
- 実際の支援判断は、専門家・現場判断を最優先してください
- 表示されるデータはすべて架空のサンプルデータです

---

## アプリ概要

本アプリは以下を目的としています：

- 過去データ（頻度）と主観的緊急度を分離して扱う
- ベイズ統計により「不確実性」を明示する
- 支援判断を **数値で補助** するが **決定は人が行う** という設計思想を示す

---

## 使用している指標

### 1. 事後平均（Posterior Mean）
過去データから推定される「支援が必要となる確率」

### 2. 94% HDI（Highest Density Interval）
不確実性を含めた確率区間  
→ データが少ない場合、区間は広く表示される

### 3. 優先度スコア（参考指標）
以下の加重平均で算出：

- 事後平均：60%
- 主観的緊急度：40%

※ あくまで参考値です
